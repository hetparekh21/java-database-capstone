# User Stories

## Admin User Stories
1. As an admin, I want to manage user roles and permissions so that I can control access to different parts of the system.
2. As an admin, I want to view system logs and user activity so that I can monitor for any suspicious behavior.
3. As an admin, I want to configure system settings (e.g., email notifications, appointment reminders) so that I can tailor the system to our needs.

## Patient User Stories
1. As a patient, I want to book an appointment with a doctor so that I can receive medical care.
2. As a patient, I want to view my appointment history so that I can keep track of my medical visits.
3. As a patient, I want to receive reminders for upcoming appointments so that I don't forget them.

## Doctor User Stories
1. As a doctor, I want to manage my availability so that patients can only book appointments when I'm free.
2. As a doctor, I want to view patient medical histories so that I can provide better care.
3. As a doctor, I want to update my profile information (e.g., contact details, specialties) so that patients have the most current information.
